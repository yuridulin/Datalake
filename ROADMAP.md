### Datalake

* Добавление тега в объекты со стороны тега
* Тип данных "время"
* Преобразование для типа "дискрет" - выбор значения из пары (0/1)
* Поисковая строка в списке значений с источника и кнопка "Создать все отображаемые теги"
* Копирование объектов (включая теги)
* Движок запросов, функционал для управления запросами
* Обработка событийных данных (либо как теги, либо как отдельная сущность events)
* Кэширование времени обновления конфигурации, уменьшение количества подключений к базе данных
* Расширение визуализации запросов (несколько типов графиков, сводные таблицы)
* Фильтр для просмотра логов
* Ограничение доступа, авторизация пользователей
* Экспорт/импорт конфигурации
* Версионирование (версия приложения, миграция схемы базы данных)
* Расширение списка источников (Datalake, Logger)

### iNOPC

* Error: Сохраняется абсолютный путь к драйверу
* Веб-консоль на React
* Драйвер, пробрасывающий данные с удалённого iNOPC
* Расширение взаимодействия с Datalake (предоставление служебной информации о драйверах и настройках значений - тип, скорость обновления и т.д.)
* Экспорт/импорт конфигураций

### Logger

* Оптимизация используемого SQL кода
* Веб-консоль на React

### Other

* Объединение взаимодействия со всеми приложениями в рамках одной консоли (сквозные запросы либо новая консоль)
